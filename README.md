# IAb_optimizer
- Version
  - 1.0.0

- Description 
  

- Input formats
  - Multi-line inputs.
  - Inputs are not case sensitive (can be in upper or lower case).
  - Each line must be a 9-mer or more.
  - Input must not contain B J O U X Z or ! @ # $ % ^ & * ( ) , numbers, line break or other special characters.
  
  

- Output formats
  - Sequence number 
  - 9mer start Pos
  - 9mer
  - 9mer aa value 
  - Possible Max Difference
  - Max Score
  - 9 mer position 
  - Original aa
  - Target aa
